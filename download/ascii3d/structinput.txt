     Structure of command lines:
     ~~~~~~~~~~~~~~~~~~~~~~~~~~~
     
     Every command does not have specific characters everywhere.
     There are special structures for the commands.
     Explanation of structure list:
     - * means any string can come and will not cunfuse the program
         (can also be empty)
     - ? is a placeholder for one character that is for better view but
         must be any character.
     - # is one digit of a three-digit-number that gives a value to the program
     - A-Z, a-z (letter) is the letter shown to perform the specific command
     - % is any character except q, s, p or j (mostly ' is used)
     - @ is a material-specific character (see list below)
     
     Create rectangular prism:		q?###?###?###?###?###?###?@*
     Print out string on screen:	s??*?
     Change position:			p?###?###?###*
     Change jumping length:		j?###*
     Comment:				%*
     End of command list:		END


     Material index list:
     ~~~~~~~~~~~~~~~~~~~~
     

     Specific
     Character	Material
     --------------------
     ' '	Air
     'h'	Wood
     's'	Stone
     'e'	Earth
     'g'	Grass
     'w'	Water
     'l'	Lava
     'n'	Snow
     'd'	Sand
     'b'	Solid
